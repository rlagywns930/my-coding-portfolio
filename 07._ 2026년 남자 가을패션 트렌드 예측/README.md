# 👔 2026년 가을 남성 패션 트렌드 예측 모델 프로젝트

> **📅 프로젝트 기간:** 2025년  
> **🎯 목표:** 2025년 네이버 블로그 데이터를 기반으로 2026년 가을 남성 패션 트렌드를 예측하는 데이터 기반 분석 프로젝트

---

## 📌 프로젝트 개요

2025년 가을에 네이버 블로그에 게시된 남성 패션 관련 콘텐츠를 수집하고, 텍스트 기반 전처리 및 키워드 분류, 피처 엔지니어링, 머신러닝 기반 예측 모델링을 수행하였습니다.  
이를 통해 2026년 가을에 어떤 트렌드가 유행할지에 대한 **정성적·정량적 인사이트**를 도출하였습니다.

---

## 📁 폴더 구조
| 폴더명           | 설명                                                        |
| ------------- | --------------------------------------------------------- |
| `1_데이터 수집`    | 네이버 블로그 API 기반 남성 가을 패션 데이터 수집 및 원본 CSV 저장 |
| `2_EDA`       | 데이터 기초 구조 파악, 결측치 및 중복 데이터 탐색             |
| `3_전처리`       | KoNLPy(Okt) 기반 명사 추출, 불용어 제거 및 텍스트 정제            |
| `4_전처리 후 EDA` | 그룹별 워드클라우드, 그룹 간 상관관계 분석(Heatmap), 키워드 분포 확인  |
| `5_피처엔지니어링`   | TF-IDF 벡터화, 3+1 그룹 재편성, LabelEncoder 저장(le_keyword.pkl)   |
| `6_모델링`       | LDA 토픽 모델링 수행 및 머신러닝 모델(RF, XGBoost) 기본 학습 |
| `7_모델 튜닝`     | GridSearchCV 기반 하이퍼파라미터 최적화 및 최종 예측 결과 도출  |

## 📊 사용 데이터

- **플랫폼:** 네이버 블로그 (Naver Blog Search API 활용)
- **데이터 성격:** 2025년 가을 시즌 남성 패션 관련 소셜 텍스트 데이터
- **수집 기간:** 2025년 9월 ~ 11월 (가을 트렌드 집중 기간)
- **주요 수집 키워드:**
  - `남자 가을 코디`, `가을 남성 패션`, `버건디 니트`, `데님 셋업`, `스웨이드 자켓`, `윈드브레이커` 등
- **데이터 규모:** 약 6,000건 이상의 포스팅 텍스트 (제목 및 본문 요약)

---

## 🧹 데이터 전처리 (Data Preprocessing)

비정형 소셜 데이터를 모델 학습에 최적화된 형태로 정제하였습니다.

* **노이즈 제거:** `BeautifulSoup` 및 정규표현식을 사용하여 HTML 태그와 특수문자 완벽 제거
* **텍스트 정규화:** `normalize_text` 함수를 통해 구어체 어미를 분석용 기본형으로 통일
* **불용어 정교화:** 광고성 문구, 연예인 이름 등 **1,000개 이상의 커스텀 불용어** 리스트 적용
* **중복 토큰 최적화:** 문장 내 반복되는 단어를 순서대로 제거하여 키워드 밀도 강화
* **데이터 무결성:** 정제 후 발생한 빈 행(NaN) 및 중복 게시글을 삭제하여 데이터 품질 확보
---

## 📈 전처리 후 EDA (Exploratory Data Analysis)

정제된 데이터를 다각도로 시각화하여 패션 트렌드에 대한 핵심 인사이트를 도출했습니다.

* **데이터 무결성 검증:** 전처리 후 결측치 및 중복 데이터를 최종 점검하여 분석 데이터의 신뢰성 확보
* **그룹별 워드클라우드:** `WordCloud`를 통해 각 패션 그룹별 핵심 키워드 비중을 직관적으로 시각화
* **키워드 상관관계 분석:** - `KoNLPy(Okt)`를 활용한 명사 추출 및 그룹별 키워드 출현 행렬 생성
  - **상관관계 히트맵(Heatmap)**을 통해 그룹 간 패션 유사도 수치화 및 분석
* **Top 10 키워드 분포:** 각 그룹별 상위 키워드 빈도를 막대그래프로 비교하여 그룹별 아이템 특징 파악

---

## 🔧 피처 엔지니어링 (Feature Engineering)

* **레이블 최적화:** 기존 그룹 데이터를 분석 목적에 맞는 3+1 체계로 재매핑
* **범주형 인코딩:** `LabelEncoder`를 통한 그룹(Group) 및 키워드(Keyword) 수치 변환
* **텍스트 벡터화:** `TfidfVectorizer`를 활용하여 텍스트 데이터를 300차원의 수치 피처로 변환
* **데이터 통합:** 인코딩 데이터와 TF-IDF 벡터를 결합하여 최종 학습용 데이터셋 구축

---

## 🤖 모델링 (Modeling)

추출된 피처와 토픽 데이터를 결합하여 패션 키워드를 예측하는 다중 분류 모델을 구축하였습니다.

### 1. LDA(Latent Dirichlet Allocation) 토픽 모델링
* **텍스트 잠재 의미 추출:** `CountVectorizer`를 통해 단어 빈도 행렬을 생성하고, LDA 알고리즘을 적용하여 데이터 내 10개의 잠재 토픽을 도출했습니다.
* **피처 엔지니어링 확장:** 문서별 토픽 분포(Topic Distribution)를 수치화하여 분류 모델의 입력 피처(Feature)로 결합함으로써 모델의 맥락 이해도를 높였습니다.

### 2. 머신러닝 분류 모델 구축 (Keyword Classification)
데이터셋을 학습용(80%)과 검증용(20%)으로 분리하여 **대표 키워드(`keyword`)**를 예측하는 모델을 학습시켰습니다.
* **Random Forest:** 다수의 결정 트리를 활용한 앙상블 학습으로 데이터의 비선형적 관계를 학습하고 안정적인 성능을 확보했습니다.
* **XGBoost:** 그래디언트 부스팅 기반의 고성능 알고리즘을 사용하여 복잡한 패션 키워드 간의 패턴을 정교하게 분류했습니다.

### 3. 모델 평가 및 복원
* **성능 지표:** Accuracy 및 Classification Report(Precision, Recall, F1-score)를 통해 모델의 예측력을 다각도로 검증했습니다.
* **레이블 복원:** `Pickle`로 저장된 `LabelEncoder`를 활용하여 수치형 예측 결과를 실제 한글 키워드 명칭으로 역변환하여 결과를 확인했습니다.

## 🧾 예측 결과 및 트렌드 인사이트 (Results & Insights)

단순 정확도 수치보다 **비정형 패션 데이터에서의 트렌드 분류 및 구조화 성능**에 집중하여 분석을 진행했습니다.

### 1. 모델 성능 및 최적화
* **XGBoost:** Accuracy **24.7% → 29.0%** (최적화 후 4.3%p 상승)
* **Random Forest:** Accuracy **23.9% → 26.8%** (최적화 후 2.9%p 상승)
* **결과:** 수백 개의 세부 키워드를 분류하는 고난도 태스크에서 하이퍼파라미터 튜닝을 통해 성능을 개선했습니다.

### 2. 정확도 지표에 대한 분석적 해석
* **데이터의 중의성:** 패션 포스팅 특성상 여러 아이템이 동시에 언급되므로, 수치적 일치보다 **맥락적 분류 타당성**이 더 중요합니다.
* **트렌드 가이드:** 100% 정답 예측이 아닌, 방대한 데이터 속에서 **지배적인 패션 흐름(버건디, 레트로 등)을 포착**하는 도구로서의 가치를 확인했습니다.

### 3. 주요 도출 트렌드
* **핵심 키워드:** '남자 데님 셋업', '레트로 패션', '캠퍼스룩' 등 테마 중심의 분류 성공
* **컬러 인사이트:** '버건디', '브라운' 등 시즌 트렌드 컬러의 언급 비중을 정량적으로 산출

### 4. 기대효과
* LDA와 ML 분류 모델을 결합하여 소셜 미디어 내 **패션 키워드 자동 분류 체계** 구축
* 데이터 기반의 시즌 주력 상품 선정 및 마케팅 전략 수립의 근거로 활용 가능
