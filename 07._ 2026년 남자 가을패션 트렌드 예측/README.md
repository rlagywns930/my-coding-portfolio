# 👔 2026년 가을 남성 패션 트렌드 예측 모델 프로젝트

> **📅 프로젝트 기간:** 2025년  
> **🎯 목표:** 2025년 네이버 블로그 데이터를 분석하여 2026년 가을 남성 패션 트렌드를 예측하는 데이터 기반 인사이트 도출

---

## 📌 프로젝트 개요
본 프로젝트는 2025년 가을 네이버 블로그의 남성 패션 콘텐츠를 수집하고, 텍스트 전처리·피처 엔지니어링·머신러닝 모델링을 거쳐 차기 시즌의 트렌드를 예측합니다. 단순한 수치 예측을 넘어, 비정형 데이터에서 **정성적·정량적 트렌드 패턴**을 구조화하는 데 중점을 두었습니다.

---

## 📁 폴더 구조
| 폴더명 | 설명 |
| :--- | :--- |
| **1_데이터 수집** | 네이버 블로그 API 기반 데이터 수집 및 원본 CSV 저장 |
| **2_EDA** | 결측치 탐색 및 데이터 기초 구조 파악 |
| **3_전처리** | BeautifulSoup 및 정규표현식 기반 텍스트 정제, 불용어 처리 |
| **4_전처리 후 EDA** | KoNLPy(Okt) 기반 명사 추출, 워드클라우드 및 상관관계(Heatmap) 분석 |
| **5_피처엔지니어링** | TF-IDF 벡터화, 3+1 그룹 재편성, LabelEncoder 저장 |
| **6_모델링** | LDA 토픽 모델링 수행 및 ML(RandomForest, XGBoost) 모델 구축 |
| **7_모델 튜닝** | GridSearchCV 기반 최적화 및 최종 트렌드 예측 결과 도출 |

---

## 📊 사용 데이터
- **플랫폼:** 네이버 블로그 (Naver Blog Search API)
- **데이터 성격:** 2025년 9월 ~ 11월(가을 시즌) 남성 패션 관련 소셜 텍스트
- **데이터 규모:** 약 6,000건 이상의 포스팅 (제목 및 본문 요약)
- **주요 키워드:** `남자 가을 코디`, `버건디 니트`, `데님 셋업`, `스웨이드 자켓`, `윈드브레이커` 등

---

## 🧹 데이터 전처리 (Data Preprocessing)
비정형 데이터를 모델 학습에 최적화된 형태로 정제하였습니다.
* **노이즈 제거:** HTML 태그 및 특수문자 제거, 구어체 어미를 분석용 기본형으로 정규화
* **커스텀 불용어:** 광고성 문구 및 불필요한 고유명사 등 **1,000개 이상의 불용어 리스트** 적용
* **토큰 최적화:** 문장 내 반복 단어 제거를 통한 키워드 밀도 강화 및 데이터 무결성 확보

---

## 📈 전처리 후 EDA
정제된 데이터를 시각화하여 패션 트렌드의 핵심 맥락을 짚어냈습니다.
* **워드클라우드:** 그룹별 핵심 키워드 비중 시각화
* **상관관계 히트맵:** 그룹 간 패션 키워드 유사도 및 출현 행렬 분석
* **분포 분석:** 상위 10개 키워드 빈도 비교를 통한 아이템 특징 파악

---

## 🔧 피처 엔지니어링 (Feature Engineering)
* **레이블 최적화:** 분석 효율을 위해 기존 그룹을 3+1 체계(아이템, 스타일링, 트렌드 컬러 등)로 재매핑
* **텍스트 수치화:** `TfidfVectorizer`를 활용하여 통합 텍스트를 300차원의 피처로 변환
* **범주형 변수 처리:** `LabelEncoder`를 사용하여 그룹 및 키워드 데이터를 수치화 후 병합

---

## 🤖 모델링 (Modeling)
### 1. LDA(Latent Dirichlet Allocation) 토픽 모델링
* **잠재 토픽 추출:** 10개의 잠재 토픽을 도출하여 문서별 분포를 수치화
* **피처 확장:** 도출된 토픽 분포를 분류 모델의 추가 피처로 활용하여 맥락 이해도 향상

### 2. 머신러닝 분류 (Keyword Classification)
* **Random Forest & XGBoost:** 8:2 학습/검증 분리를 통해 대표 키워드(`keyword`) 예측
* **레이블 복원:** `Pickle`로 저장된 Encoder를 사용해 예측 결과를 실제 한글 명칭으로 역변환

---

## 🧾 예측 결과 및 분석 (Results)
### 1. 모델 성능 및 최적화
* **XGBoost:** Accuracy **24.7% → 29.0%** (최적화 후 4.3%p 향상)
* **Random Forest:** Accuracy **23.9% → 26.8%** (최적화 후 2.9%p 향상)

### 2. 수치 지표에 대한 분석적 해석
* **정답의 다중성:** 패션 포스팅은 여러 아이템이 공존하므로, 단순 일치율보다 **맥락적 분류 타당성**이 더 중요함
* **트렌드 가이드:** 100% 정답 맞히기보다, 방대한 데이터 속 지배적인 **패션 흐름(버건디, 레트로 등)을 포착**하는 데 성공

### 3. 2026년 예상 트렌드 키워드 인사이트
| 트렌드 유형 | 예측된 키워드 인사이트 예시 |
| :--- | :--- |
| **컬러** | `버건디`, `톤온톤`, `뉴트럴톤` 중심의 스타일링 지속 |
| **스타일** | `캐주얼`, `꾸안꾸`, `미니멀`, `셋업`, `윈드브레이커`, `레더 자켓` 등 |
| **TPO** | `캠퍼스룩`, `데이트룩`, `출근룩`, `스트릿 패션` 등 |
| **아이템** | `셔츠`, `데님`, `트랙셋업`, `가디건`, `모자` 등 |

## 📌 주요 인사이트 (Key Insights)
* **카테고리별 체계적 분류:** 색상(Color), 스타일(Style), 아이템(Item), 상황(TPO)의 4가지 축을 중심으로 트렌드 군집화
* **스테디셀러의 지속성:** `레더 자켓`, `데님 셋업` 등은 높은 예측 확률을 보이며 2026년에도 핵심 아이템으로 작용할 전망
* **Z세대 트렌드 확산:** `꾸안꾸`, `고프코어` 등 스타일링 중심 키워드 비중 상승으로, 단일 제품보다 '착장 테마' 중심 콘텐츠 선호 경향 확인

## 🔮 한계 및 향후 방향
* **데이터 레이블 부재:** 정답이 없는 데이터 특성상, 예측 결과의 **정성적 인사이트** 도출에 집중
* **트렌드 사후 검증:** 2026년 실제 데이터와 비교 분석(Back-testing)하여 모델 예측력 검증 예정
* **채널 확장:** 유튜브, 인스타그램 등 멀티 채널 데이터 통합 및 **LLM(GPT) 기반 자동 분류 시스템**으로 고도화 가능성
