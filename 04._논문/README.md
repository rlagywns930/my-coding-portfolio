# 📄 초음파 센서와 LSTM을 활용한 수위 예측 시스템

## ✅ 개요  
이 프로젝트는 HC-SR04 초음파 센서와 아두이노를 활용하여 물이 채워지는 수위 데이터를 수집하고, 이를 기반으로 LSTM(Long Short-Term Memory) 모델을 통해 물병이 가득 차는 데 걸리는 시간을 예측하는 실험 기반 연구입니다.

## 🧪 실험 개요  
- **센서 장비**: HC-SR04 초음파 센서, Arduino Uno  
- **측정 대상**: 25cm 투명 물병  
- **수집 데이터**: 수위 도달 시간 (1/10, 1/8, 1/6, 1/4, 1/2), 총 소요 시간  
- **데이터 전송**: 아두이노 → 시리얼 통신 → Python 수신 → CSV 저장  

## 🤖 모델 개요  
- **입력**: 5개의 수위 도달 시간 (초 단위)  
- **출력**: 물이 가득 찰 때까지의 총 소요 시간 (초 단위)  
- **모델 구조**: LSTM(64 유닛) + Dense 출력층  
- **학습**:  
  - 정규화: Min-Max Scaling  
  - 손실 함수: Mean Squared Error (MSE)  
  - 최적화 알고리즘: Adam  
  - 반복 횟수: 300 Epochs

## 📊 결과 및 의의  
- 예측 결과는 실제 측정값과 비교하여 정확도를 분석  
- 기준 시간(예: 70초)보다 짧게 예측되면 침수 위험 가능성 판단  
- **저비용 센서** + **딥러닝 모델**의 결합으로 수위 예측 자동화를 실현
